# AI 广告文案生成与 Prompt 优化工具

## 项目目标
输入一段产品信息，自动生成三种风格的广告词，并且自己设计评分标准，根据生成结果对Prompt打分。

## 产品案例
假设一个产品场景：  

- 产品：英语口语练习app  
- 目标用户：大学生  
- 投放平台：小红书  

### 一、确定输入输出
**输入**
- 产品名称
- 产品卖点
- 目标用户
- 投放平台

**输出**
- 三种不同风格的广告文案

### 二、Prompt模板
#### **Prompt A: 直接营销型**
- 强调产品功能
- 强调效率提升
- 强调行动号召

预期口号类似：“7天提升口语表达，自律者必备”
#### **Prompt B: 真实分享型**  
- 模仿普通用户分享日常
- 强调效率提升
